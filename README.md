# LLD Practice – JavaScript

Hands-on practice of core Low Level Design concepts using JavaScript.

---

## Topics Covered

### OOP
- Encapsulation, Inheritance, Polymorphism, Abstraction  

### UML
- Dependency, Association, Aggregation, Composition  
- Inheritance, Realization  

### SOLID
- SRP, OCP, LSP, ISP, DIP  

### Design Patterns

#### Behavioral
- **Memento Pattern** → undo/restore state using snapshots  

---

## Key Learnings
- Prefer abstractions over direct dependency  
- Use polymorphism over conditionals  
- Keep classes small and focused  
- Design for extension
